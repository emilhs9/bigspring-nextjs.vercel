# Bigspring

Bigspring HTML, CSS və JavaScript ilə hazırlanmış sadə və responsiv landing page layihəsidir.

## Live Demo

https://bigspring-nextjs.vercel.app

## Layihə haqqında

Bu layihədə Bigspring adlı veb saytın ana səhifəsi hazırlanıb. Saytda navbar, hero bölməsi, brend loqoları, xidmətlər, testimonial hissəsi, əlaqə bölməsi və footer mövcuddur.

## İstifadə olunan texnologiyalar

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## Xüsusiyyətlər

- Responsiv dizayn
- Mobil menyu
- Dark / light mode
- Slider bölmələri
- Video modal
- Scroll to top düyməsi
- Vercel deploy

## Layihəni işə salmaq

Layihəni lokalda açmaq üçün sadəcə `index.html` faylını brauzerdə açmaq kifayətdir.
